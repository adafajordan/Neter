# Neter

Neter is a mobile-first Progressive Web App (PWA) that acts as a hub for both news and social media.

## Features

- Mobile-friendly home screen with curated news and social links.
- Installable PWA using a web app manifest.
- Offline-capable shell powered by a service worker cache.

## Run locally

Serve the repository with any static server, for example:

```bash
cd <repository-directory>
python3 -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
